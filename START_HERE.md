# Start Habit Kingdom

## Fastest option

1. Install Node.js 20 or newer.
2. Open a terminal in this folder.
3. Run `npm install`.
4. Run `npm run dev`.
5. Open the address shown in the terminal.

## Open the production version

Run `npm run preview`, then open the shown address. A tested production build is already included in the `dist` folder.

## Share with a teacher

1. Create a GitHub repository and upload this entire folder.
2. Open **Settings → Pages** in the repository.
3. Select **GitHub Actions** under **Build and deployment**.
4. Push to `main`, or run the deployment workflow from the **Actions** tab.
5. Send the generated `https://your-name.github.io/repository-name/` link to the teacher.

The application has no server dependency and runs entirely in the browser. User progress is stored separately in each browser.
